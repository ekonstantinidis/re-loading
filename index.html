Reloading

A Loading component for React.

With a CSS Class

You can pass a CSS class by using the `className` propertly. Classes can be used with all other properties.

<Loading
  shouldShow={this.state.loading}
  className='loading-wrapper' />

With a Font Awesome Icon

Just pass the name font awesome classes through the `faIcon` property as per the example. This will not display any text.

<Loading
  shouldShow={this.state.loading}
  faIcon='fa fa-refresh fa-spin' />

With Custom Content

Passing children to the component will override using properties like text & icon.

<Loading shouldShow={this.state.loading}>
    Your content - images, text, whatever!
</Loading>

With Text

Want to only display text? Just use the `text` attribute to pass the text you would like.

<Loading
  shouldShow={this.state.loading}
  text='Checking your details...' />
